Swagger+Springmvc打造最轻松的api文档

java

简介

  • 随着项目越来越大，大家的分工越来越明确，前后端之间的沟通成本就变得越来越高。常常后端写完接口之后，还要花时间整理接口文档给前端。但接口有时候出错，或者进行了轻微的调整，那就是一件非常麻烦的事情了。不但要先更新接口文档，还要通知前端哪些要改。另外总因为文档是滞后的，所以效率变得很低。所以这时候swagger的重要性就突显出来了，它能够根据你项目每一次更新，而自动更新接口文档，并提供在线接口测试，而代码仅仅是增加几行注解这么简单！

    项目要求

  • 实际上基本上所有的项目都支持，目前仅介绍springmvc类型的项目
